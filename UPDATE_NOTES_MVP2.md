# MVP 2 update

This update fixes the two issues found during the first Order Siaw test:

1. Scroll-animated Services, Projects, Process and Why Work With Us cards are forced visible only inside the editor canvas. Their original animations still run in Live Preview and in the exported website.
2. The featured AgriSense image in “Browse products, services and order requests” is controlled by `script.js`. The editor now synchronises a replacement made to `detailImage` into the AgriSense product data when Save Changes or Live Preview is used.

It also adds cache-busting to Live Preview and editor-only notices for content generated by JavaScript.
